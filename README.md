# Orbit IP

 A IP Tracker Built on Python!

# How To Install

First, you must install it in your terminal
```bash
$ git clone https://ItsBr0dyy/Orbit-IP
```
After cloning, make sure you have all dependencies installed
```bash
cd Orbit-IP
pip install
```
Once you got all two done, then its ready for use
```bash
python orbit.py
```
